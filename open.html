OpenProcessing

OpenProcessing is website users can use to create sketches in programing language. Instead of supporting all the programming languages, OpenProcessing focuses on Javascript and p5js. On this page, I have added all of the sketches I have created using OpenProcessing.

Sketch 1 - Hello World!

Hello World! Sketch

This sketch was created in my SMAD 308: Interactive Design 2 class. We were tasked to introduce ourselves through code. I choose to draw a penguin because it is one of my favorite animals, and had elements I knew how to code without any previous instruction.

Sketch 2 - Abstract Art

Abstract Art Sketch

This sketch was created in my SMAD 308: Interactive Design 2 class. We were asked to generate a visual reinterpretation of one of Mondrian's works of art using lines, shapes, and colors. Mondrian is known for being the pioneer of 20th-century abstract art, which uses simple geometric shapes.

Sketch 3 - Animation

Animation Sketch

This sketch was created in my SMAD 308: Interactive Design 2 class. We were tasked to create a sketch that includes an animation that responds to the user's mouse in some way. I chose to create a camera that flashes when you press down on the mouse. I chose this design because Photography is one of my passions and I wanted to display that passion in code.

Sketch 4 - Portrait

Portrait Sketch

This sketch was created in my SMAD 308: Interactive Design 2 class. We were tasked to design a portrait of our choosing. You could create a portrait of yourself, a friend, your dog, a celebrity, a cartoon character, etc. I decided to create one of my favorite cartoon characters, Perry The Platypus from Phineas and Ferb.

Sketch 5 - Visualizing Time

Visualizing Time Sketch

This sketch was created in my SMAD 308: Interactive Design 2 class. We were tasked to create a sketch that uniquely visualizes time. As well as use, hour(), minute(), and second() functions. I decided to create the phrase "Always moving", but have the "o" represent a clock and the background change color with the time.

Sketch 6 - Data Visualization

Data Visualization Sketch

This project was created in my SMAD 308: Interactive Design 2 class. We were tasked to create a sketch that visualizes data that is either loaded from a JSON or CSV file. I chose to make an infographic using a CSV file. The topic was "COVID-19 Vaccinated People in the USA - 2021". Once I gather my stats, I created this infographic based on a region of the US and amount of vaccinated people within that region.
